# Component Extractor

A bookmarklet tool that lets you extract HTML markup and CSS styles from any element on any webpage.

## Features

- Select any element on a webpage
- Extract HTML markup
- Extract computed CSS styles (with pseudo-classes and pseudo-elements)
- Works on any website via bookmarklet

## Using the Bookmarklet

- Drag the bookmarklet to your bookmarks bar
- Navigate to the site with the element you want to extract
- Click the bookmarklet
- Click 'Select Element' in the widget
- Click the element you want to extract
- Copy the HTML and CSS
